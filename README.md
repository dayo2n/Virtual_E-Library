# Virtual_E-Library

## 📚 Summary
- 2021학년도 1학기 (5학기) 데이터베이스 과목 개인 텀프로젝트
- 가상 온라인 도서관 시스템
- 회원들은 웹을 이용해 직접 도서관의 e-book 도서를 검색, 대출, 예약, 반납 기능 이용 가능

## 🛠 Tech
- Oracle RDBMS, sql
- php
- IDE VSCode
